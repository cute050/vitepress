#### 前端项目初始化

##### 确认node版本
nodejs版本：v18.16.0或16
检测命令
```shell
node -v
```
npm版本
```shell
npm -v
```
##### 初始化

使用vue-cli进行初始化
安装脚手架工具
```shell
npm install -g @vue/cli
```
检测是否安装成功命令
```shell
vue -V
```
创建项目
```shell
vue create yuoj-frontend
```
